# PiggyBank Protocol — Independent Public-Source Risk Review

This repository contains an independent public-source risk review of PiggyBank Protocol.

The report reviews PiggyBank Protocol as it appeared in the public record through **15 June 2026**. It examines the protocol’s vault design, pb-token accounting, off-chain NAV process, collateral routing, funding-rate strategy, external-venue dependencies, LAB incident, and transparency profile.

## Report

* [PiggyBank Protocol — Independent Public-Source Risk Review](./PiggyBank_Protocol_Independent_Public_Source_Risk_Review.pdf)

## Scope

This is a public-source due-diligence review. It is not investment, financial, legal, or forensic advice. It does not independently reconstruct all wallets, all venue positions, or all NAV marks from raw on-chain data.

Third-party allegations are treated as allegations unless independently corroborated.
